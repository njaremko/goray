- [x] Concurrency support
- [x] Ray-Sphere intersection
- [x] Ray-Triangle intersection
- [x] Directional lighting
- [x] Diffuse lighting
- [ ] Reflections (Almost done)
- [ ] Refraction (Almost done)
- [ ] Fresnel (Almost done)
